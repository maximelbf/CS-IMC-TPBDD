Question 3 : 
Trouve les noeuds correspondant à l'artiste "Maxime Leboeuf" et au film "L'histoire de mon 20 au cours Infrastructure de donnees"
et crée la relation "acted_in" entre ces derniers
MATCH (max:Artist), (inf:Film)
WHERE max.primaryName = 'Maxime Leboeuf' AND inf.primaryTitle = "L'histoire de mon 20 au cours Infrastructure de donnees"
CREATE (max)-[r:acted_in]->(inf)
RETURN type(r);

Question 8 : 
Récupère tous les artistes ayant une relation acted_in et compte le nombre de films par acteur.
Filtre ensuite l'aggrégation pour trouver les artistes ayant joué dans plus d'un film,
et affiche le nom de ces artistes avec le nombre de films dans lequel ils ont joué.
MATCH (a:Artist)-[r:acted_in]->(f:Film)
WITH a, count(DISTINCT f.idFilm) AS nbFilms WHERE nbFilms > 1
RETURN a.primaryName, nbFilms

Question 9 :
Récupère toutes les relations des artistes et compte les relations de type distincts.
Affiche ensuite les artistes ayant eu plus d'une seule responsabilité dans leurs vies.
MATCH (a:Artist)-[r]->()
WITH a, count(DISTINCT type(r)) AS respos WHERE respos > 1
RETURN a.primaryName, respos;

Question 10 :
Principe similaire à la précédente, avec l'ajout des films dans l'aggrégation, ce qui permet de compter les responsabilités
film par film.
Affiche ensuite le nom des artistes et le titre des films dans lesquels ils ont eu plusieurs responsabilités.
MATCH (a:Artist)-[r]->(f:Film)
WITH a, f, count(DISTINCT type(r)) AS respos WHERE respos > 1
RETURN a.primaryName, f.primaryTitle, respos;

Question 11 :
Compte le nombre d'acteurs par film puis trouve le maximum de cette valeur.
Filtre ensuite les films sur leur nombre d'acteurs pour trouver deux correspondant au maximum.
Affiche le nom des films avec le nombre d'acteurs.
MATCH (a:Artist)-[:acted_in]->(f:Film)
WITH f, count(DISTINCT a) AS nbActor
WITH max(nbActor) AS maxActors
MATCH (a:Artist)-[:acted_in]->(f:Film)
WITH f, count(DISTINCT a) AS nbActor, maxActors
WHERE nbActor = maxActors
RETURN f.primaryTitle, nbActor;
